# منصة مشروعي - النسخة المعاد بناؤها

هذا المستودع يحتوي على نسخة تشغيلية مبسطة ومصححة من منصة **مشروعي**، مع طبقة محاكاة محلية تعمل دون Backend خارجي.

## حالة الرفع

تم تهيئة المستودع `yalnajjar2/2026` ورفع نسخة تشغيلية مباشرة قابلة للفتح من المتصفح.

> ملاحظة: النسخة الكاملة التي تحتوي على الصور والخطوط وكل الملفات الأصلية موجودة كملف ZIP تم تجهيزه داخل المحادثة باسم:
>
> `mashrooi-platform-rebuilt-fixed.zip`

## التشغيل المحلي

افتح الملف التالي مباشرة في المتصفح:

```text
index.html
```

أو شغّل خادمًا محليًا:

```bash
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```
